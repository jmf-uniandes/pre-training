# Markdown keywords - G2 -

## Elementos de Bloque

### 1. Encabezados (Títulos)
Se utilizan los signos de número (`#`) para definir niveles de título.

| Nivel | Sintaxis |
| :---: | :---: |
| H1 | `# Título Principal` |
| H2 | `## Subtítulo` |
| H3 | `### Título Menor` |

### 2. Párrafos y Saltos de Línea
Los párrafos se separan dejando una línea en blanco.

### 3. Listas
- **No ordenadas:** Utiliza `*`, `-`, o `+`.
    * Ejemplo: `* Item uno`
- **Ordenadas:** Utiliza números seguidos de un punto.
    * Ejemplo: `1. Primer item`

### 4. Citas en Bloque
Se inicia la línea con el signo "mayor que" (`>`).

> Esto es una cita en bloque, útil para destacar texto.

### 5. Bloques de Código
Se usan tres acentos graves (```) en líneas separadas.

### 6. Enfasis en fuente
Negrita  **Texto Negrita**
Cursiva *Texto Cursiva*
Tachado ~~Texto Tachado~~
### 7. Tablas
Se crean utilizando tuberías (`|`) y guiones (`-`).
| Encabezado 1 | Encabezado 2 |
| :-----------: | :-----------: |
|    Dato 1    |    Dato 2    |
|    Dato 3    |    Dato 4    |

### 8. Saltos de Línea Forzados
Se pueden insertar dos espacios al final de una línea para forzar un salto de línea.

### 9. Línea Horizontal
Se crean con tres o más guiones (`---`), asteriscos (`***`), o guiones bajos (`___`) en una línea separada.
___

### 10. Comentarios en Markdown
Los comentarios no se muestran en el renderizado final y se crean utilizando la sintaxis HTML:
<!-- Este es un comentario en Markdown -->

### 11. Referencias de Enlaces
Se pueden definir enlaces con referencias para mayor claridad.
[Google]: https://www.google.com "Ir a Google"
[Ejemplo de enlace a Google][Google]

### 12. Imágenes
Se insertan utilizando una sintaxis similar a la de los enlaces, pero precedida por un signo de exclamación (`!`).
![Texto Alternativo](ruta/a/la/imagen.jpg "Título de la Imagen")
