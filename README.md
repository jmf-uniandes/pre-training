# Case-study-seminar-G2

> Este proyecto aborda el desafío de predecir el éxito de una canción utilizando datos históricos de Spotify. A través del **Análisis Exploratorio de Datos (EDA)** y el **Aprendizaje Automático (Machine Learning)**, el proyecto tiene como objetivo descifrar la "fórmula secreta de un hit".

## Nombre Caso de Estudio

#### **Análisis de Atributos Musicales y Predicción de Popularidad de Canciones.**

---

### Integrantes

- Edison Gonzalez Alberca
- Jose Miguel Figueroa Chilan
- Carlos Pintag Quispe

---

## Objetivos

Construir un sistema que clasifique las canciones en "éxito" (hit) o "no éxito". El proyecto se enfocará en el Análisis Exploratorio de Datos (EDA) para descubrir la "fórmula de un hit" (ej. ¿las canciones populares son más rápidas y bailables?) y en entrenar un modelo de clasificación para predecir la popularidad de una nueva canción a partir de sus características sonoras.
